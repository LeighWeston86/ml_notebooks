Direct-Indirect Classifier¶

Leigh Weston, August 2017¶

Quaternary semiconductors with a zinc blende kesterite structure are promising solar absorbers for photovoltaics applications. Whether or not the band gap is direct or indirect is a fundamental property that can affect device performance.

Direct materials, while typically being stronger absorbers, have shorter carrier lifetimes and suffer from carrier recombination, and this negatively impacts efficiency in solar cells.

The best way to determine if a material is direct or indirect, is to perform high-level quantum mechanical calculations on the electronic structure. However, there are 1568 possible kesterite semiconductors, and it would be far too computationally expesnsive to perform accurate band structure calculations for all materials.

I have calculated the band gaps for a subset (n = 200) of these mataerials, and use the resutls to train a direct-indirect classifier that predicts whether a material is direct or indirect. The model is then used to make predictions for all 1568 materials.

Feature selection¶

The results for a 3rd order polynomial indicated overfitting. One way to overcome this, while still having the advantage of keeping some higher-order terms, is to use feature selection.

I take the feature space, which contains over 800 features per compound, and use recursive feature elimination to prune features of a low importance, leaving a mcuh smaller feature space,however with features with a degree of importance.

It is not known $a\ priori$ how large of a subset of we should select. The code below loops over possible subset sizes, and does a grid search on the new feature space to optimize a new classifier; the best $C$ parameter and best classification accuracy score are reported.

In [18]:
df_poly3 = pd.DataFrame(X_poly3)
params_fs = {'C': np.logspace(-6, 0, 7)}
best_score = 0
for num_features in np.arange(10, 110, 20):
    rfe_new = RFE(LogisticRegression(), num_features)
    fit_features = rfe_new.fit(X_poly3, y)
    
    new_features = pd.DataFrame()
    for feature in np.where(fit_features.ranking_ == 1):
        new_features = pd.concat([new_features, df_poly3.iloc[:, feature]], axis = 0)
    
    X_fitted = new_features.as_matrix()
    grid = GridSearchCV(LogisticRegression(), params_fs, cv = 10)
    results_search = grid.fit(X_fitted, y)
    
    print('Grid search best params for {} features: {}'.format(num_features, results_search.best_params_))
    print('Grid search best score for {} features: {}'.format(num_features, results_search.best_score_, sep = '\n'))
    
    if results_search.best_score_ > best_score:
        best_score = results_search.best_score_
        best_params = results_search.best_params_
        best_feature_size = num_features
        best_features_array = np.where(fit_features.ranking_ == 1)
        best_features = X_fitted
        
print('The best score is: {}'.format(best_score))
print('The best C parameter is: {}'.format(best_params))
print('The best size for the feature set is: {}'.format(best_feature_size))     
print('Vector of best features: {}'.format(best_features_array))
print('The best features for the training data have been saved to the variable "X_fitted"')

Using a combination of feature engineering and feature selection, I was able to significantly improve the accuracy of the classifier. Without feature selection, the classifier had an accuracy score of 81.7%; by pruning all but the 30 best features, I was able to acheive an accuracy score of 88.7%.

I did tests for higher order polynomials (not shown here, I had to do them on a supercomputing cluster to prune such a a large feature space).

89% is the best I can achieve; better results would most likely require a larger training set.

Conclusion¶

A direct-indirect classifier for I2-II-IV-VI4 semiconductors was developed using logistic regression. The simple 15-D feature space did not provide adequate predictions on the nature of the gaps. Using feature engineering within polynomial features, and feature extraction, the best classfier had an accuracy of 89%. Typically, around 70% of the materials are found to be indirect; however, this depended on the anion in the compound.
